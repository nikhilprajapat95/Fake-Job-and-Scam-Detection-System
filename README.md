# AI-Based Fake Job & Scam Detection System

## 📌 Problem Statement
With the rapid growth of online job platforms, many job seekers—especially students—fall victim to fake job postings and fraudulent internships. These scams often lead to financial loss, data theft, and wasted time.

## 💡 Solution
This project presents an AI-powered system that analyzes job descriptions or URLs and predicts whether a job posting is real or fraudulent.

## 🔍 Features
- ✅ Scam Probability Score (0–100%)
- ⚠️ Risk Level (Low / Medium / High)
- 🧠 NLP-based Text Classification
- 📊 Feature Engineering (salary, keywords, etc.)
- 🤖 AI-generated Explanation (GenAI)
- 🔍 Suspicious Keyword Highlighting
- 🌐 URL Analysis (optional)

## 🧠 Tech Stack
- Python
- Flask
- Scikit-learn
- NLP (TF-IDF / BERT)
- HTML, CSS, JavaScript
- OpenAI / Gemini API (for explanation)

## 📂 Project Structure
- Data Ingestion & Cleaning
- Feature Engineering
- Model Training & Evaluation
- Prediction Pipeline
- Web Application (Flask)

## 📈 Use Cases
- Students verifying job postings
- Detecting internship scams
- Online job platform moderation

## 🚀 Future Scope
